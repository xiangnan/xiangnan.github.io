git及repo常用命令（to be continue）

Posted on 2019-05-03   |   In Tools   |     |  

git及repo常用命令记录（to be continue）。

1、下载代码

git clone

2、查看本地及远程分支
3、显示当前的Git配置(查看代码地址)
$ git config --list
4、删除分支

删除本地分支（需先切到其他分支）

git branch -d 

git branch -d dev

删除远程分支

git push origin –delete 

git push origin --delete feature1
5、选择一个提交合并到当前分支
$ git cherry-pick cb84671
6、git fetch及git pull

git fetch 默认取回所有分支更新

若取回特定分支的更新
git fetch <远程主机名> <分支名>

git fetch origin master //取回origin主机的master分支

git pull = git fetch + git merge

若不想用merge形式，可指定为rebase形式 git pull –rebase
